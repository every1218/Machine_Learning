**문제 1**

**문제 2**

기말고사 점수는 `60, 87, 55, 47, 92`이다.  
중간고사 평균과 기말고사 평균을 각각 구하시오.

**문제 3**

중간고사와 기말고사 평균을 각각 구하시오.

**문제 4**

중간고사는 50%, 기말고사는 30%의 비중을 두어 성적을 산출한다.  
각 학생의 최종 성적을 구하고, 이를 기준으로 등수를 나열하시오.
